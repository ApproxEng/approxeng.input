PlayStation 4 Controller Support¶

DualShock4 Controller

Nothing more to do for this controller, it pairs over Pairing Controllers over Bluetooth with no issues. This library doesn’t provide any support for the motion sensing, or for the trackpad (other than using it as a button). This is partly because it’s the only controller with those controls, and partly because only some sub-versions of the controller hardware actually work with the current Linux driver (mine, for example, doesn’t).
