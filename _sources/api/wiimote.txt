WiiMote Controller Support
==========================

Version 1.0.7 adds support for the WiiMote along with the optional Classic or Nunchuck controllers, both of which
connect to the WiiMote wand, so you can have either the Classic or the Nunchuck but not both!

Support contributed by Keith Ellis
